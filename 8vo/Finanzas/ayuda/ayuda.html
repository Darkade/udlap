Bienvenido al valuador de opciones

Bienvenido, el valuador de opciones ha sido desarrollado para asistirle al valuar opciones mediante el método binomial. Este software ha sido desarrollado suando Python y GTK+ y se distribuye bajo la licencia GPL, esta licencia se encuentra disponible en http://www.gnu.org/licenses/gpl.html Este programa es software libre

Datos de entrada Válidos

La entrada de datos al software es mediante cajas de texto, estas cajas no solo aceptan número sino expresiones matemáticas como: 2/3 o 10-3



Los campos aceptan las siguientes entradas

  1. u: coeficiente de alza de precio: Que tanto se alza el precio por cada paso en el arbol
  2. d: coeficiente de baja de precio: Qué tanto baja el precio por cada paso
  3. r: Tasa libre de riesgo expresada anualmente
  4. x: Precio de ejercicio para la opción
  5. Precio Spot: El precio al día de hoy del activo
  6. Tiempo: El tiempo hasta la expiración de la opción, expresada en años. Seis meses se pueden expresar 1/2,6/12, etc.
  7. Niveles: Para cuantos niveles se va a manejar el arbol

Limitaciones del programa

La única limitación que se encontro en tiempo de desarrollo fue para niveles mayores a 1029 pasos, es decir, el software no evaluará opciones si se le pide un arbol de más de 1029 niveles

El código fuente

El codigo fuente esta disponible en la carpeta "Código fuente" son archivos .py, es decir de fuente de Python. Estos archivos se deben de poder abrir en cualquier editor de texto que soporte "Plain Text Files"

Cómo funciona?

El software esta difivido en dos partes, una librería que realiza todo el trabajo, y una interfaz grafica, que simplemente muestra el formulario y se conecta con la librería

ya que en windows Python no esta disponible se debio compilarlo usando py2exe, el cual exporta todas las dependencias necesarias para el programa. es por esto que son tantos archivos cuando normalmente solo serían los que estan incluidos en la carpeta de codigo fuente

De Python

Python es un lenguaje de programación libre y de codigo abierto cuyo lema es "Batteries Included" esta disponible (preinstalado) en todos los sistemas operativos basados en Unix (Mac OS X, Linux, BSD) y para Windows mediante instalación explicita en 32 bits.
